# Week 1 Summary — Python Basics + Basic SQL

## Python — key takeaways
- **Variables / types:** int, float, str, bool
- **Flow control:** if/elif/else; while & for loops; break (exit) vs continue (skip); range()
- **Functions:** def, parameters vs arguments, return (gives value back) vs print (just shows); local scope; None
- **Error handling:** try/except — catch errors instead of crashing
- **Lists:** ordered collection; index from 0; negative index = from end; slicing [start:stop]; methods append/remove/sort/len; `in` for membership
- **Misc:** import (random), f-strings f"...{var}...", input()→int() conversion

## SQL — key takeaways
- **SELECT** chooses columns | **FROM** picks table | **WHERE** filters rows | **ORDER BY** sorts (ASC default, DESC reverse)
- **Fixed order:** SELECT → FROM → WHERE → ORDER BY (can't reorder)
- Text in quotes: `WHERE name = 'Bhagat'`; combine conditions with AND / OR
