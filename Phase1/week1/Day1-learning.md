Basic Data Type:-

1- Integer -> -2, -1, 0, 1, 2

2- Float -> -1.4 , 0.0, 3.6

3- String -> "Hello", 'hello'

4- Bool -> True , False



Type Conversion -> Changing data type of value as required. Ex. str(),int(),float(),bool():-

1- Implicit -> Done by Python itself

2- Explicit -> Done by programmer



variable-> box in the memory that keep value. Variable's value can be override.

Rules for variable naming :-

* It can’t have spaces.
* It can use only letters, numbers, and the underscore (\_) character.
* It can’t begin with a number.
* It can’t be a Python keyword, such as if, for, return, or other keywords you’ll learn in this book.
